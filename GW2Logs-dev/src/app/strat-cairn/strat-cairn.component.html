Mécaniques de l'arène
    Mécaniques de base
    • Unseen Burden - Ralentit la cible de 1% par stack. Stack max de 99.
    • Cosmic Aura - Aura qui endommage les joueurs.
    • Action Spéciale - Une action spéciale permet de se se déplacer de la même manière que le 3 du bâton du druide.
    • Shared Agony - Toutes les 20 secondes, le joueur le plus éloigné, n'ayant pas déjà le debuff, est choisi par le boss. 3 joueurs peuvent avoir le debuff en même temps. Inflige de légers dégâts sur le joueur choisi et de lourds dégâts sur une zone autour. Persiste pendant 60 secondes.
    • Displacement - Trois zones rouges pop sous le raid, les joueurs le prennant se font téléporter de manière aléatoire sur l'arène.
    • Tanking - Le boss vise le joueur le plus éloigné.
    • Energy Surge - Le boss se téléporte sur 5 endroits différents, laissant une zone verte sur son passage. Les joueurs se trouvant sur le chemin de tp se font bump. Lance Spatial Manipulation à la fin de l'attaque.
    • Meteor Swarm - Lance 8 projectiles qui bumpent les joueurs.
    • Orbital Sweep - Attaque sur 360 degrés. Les joueurs prenant le coup se font bump. Le bump est puissant et peut faire sortir le joueur de l'arène.
    • Spatial Manipulation - Inflige de gros dégâts et fait flotter le joueur. Ajoute des stacks de Unseen Burden. On l'évite en se positionnant sur une zone verte. Les petites zones requièrent 1 personne au moins pour fonctionner, les moyennes 2 et les larges 4. Si toutes les zones sont de la même taille, pas d'exigence sur le nombre de joueurs.
    • Gravity Wave - Attaque de type vague vers l'extérieur. Repousse le joueur le prennant. 3 vagues par attaque.
